##

XTEC Blocs is a blog system based on Wordpress provided by the Ministry of Education of the Government of Catalonia. It is intended to provide a platform for primary and secondary teachers and schools in order they have an official blog.


REQUIREMENTS
-----------------------------------------------------------------------------
PHP 5.3, MySQL 5.0

-----------------------------------------------------------------------------
See INSTALL.txt for installation instructions.
